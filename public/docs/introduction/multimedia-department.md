# 关于多媒体部

## 部门介绍

三院科协多媒体部，是科协的三大技术部门之一。 （多媒体部有很多好看的师兄师姐的） 

多媒体部现有的学习方向有：网站开发、UI设计、3D建模以及视频剪辑。 它旨在培养一批充满活力，具有设计美感，掌握本专业所需的基础理论、专业知识以及具有创新精神的复合型人才。

多媒体部同时也会承担一些科协内部的宣传工作。如果你想开发出自己独特的网站、剪辑出超爆款视频、设计出精美的UI以及模型，那么加入多媒体是你不二之选！
